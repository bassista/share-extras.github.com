All add-ons should work with version 3.3 and 3.4 of Alfresco Share, except where noted.

Download/Install

Each add-on is available as a separate JAR or AMP file for easy installation. To download the latest release of an add-on, see the Downloads section.

You can also build the add-ons from source using Ant.

Sample Projects

A Sample Project is provided as a basic example of how to organise your own projects, including Eclipse configuration and an Ant build script.

Contribute

You can help us build up an even bigger set of add-ons for Share by contributing your own extensions, or even just ideas for suitable extensions by filing an issue.

To discuss development of Share Extras extensions, use the Development Group.
