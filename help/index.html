Help

Share Extras is a community project and we do not provide commercial support for the add-ons listed on this site. However with the help of the community we aim to improve and expand the set of available add-ons.

Having Problems?

If you have problems with a specific add-on then please report this to us. You may find that someone else has already reported the problem in the project's issues project - if so then you can add a comment to that existing report, with any additional information you have.

When you submit an issue report, please mention

  • The version of the add-on
  • The version of Alfresco
  • Which web browser(s) does the problem occur in?

You may also want to try a more recent release of the add-on, or pull the latest from Git and build that.

Still Unsure?

If you are unsure about whether a problem you are seeing is really an issue, or if you want to discuss a wider point, please send an email to the mailing list.

Please, do not contact individual project owners or developers individually. By using the public mailing list, your questions and their replies become a resource which others can use, and you are more likely to receive a response from the group versus picking a specific individual.
